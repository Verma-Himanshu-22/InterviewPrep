# InterviewPrep

InterviewPrep is a premium AI-powered interview preparation platform.
It includes a React/Vite client and an Express backend that support resume upload,
AI-driven interview question generation, voice-based mock interviews, performance
analytics, interview history, and payment integration.

## Project Structure

- `client/` - React frontend built with Vite, Tailwind-style utility classes, and Redux.
- `server/` - Express backend with authentication, interview APIs, payment routes, and database connectivity.

## Key Features

- AI interview setup and resume analysis
- Live interview flow with speech recognition and AI voice prompts
- Performance report with analytics and downloadable PDF
- Interview history tracking
- Premium dark theme UI with updated branding to InterviewPrep

## Run Locally

1. Install dependencies for client and server:
   - `cd client && npm install`
   - `cd ../server && npm install`
2. Start client in one terminal:
   - `cd client && npm run dev`
3. Start server in another terminal:
   - `cd server && npm run dev`
